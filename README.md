# 📚 Электронный журнал оценок

Flask-приложение для учета оценок студентов с веб-интерфейсом.

## 🚀 Функции

- 📋 Просмотр всех оценок в таблице
- ➕ Добавление новых оценок
- 📊 Статистика успеваемости
- 🎨 Цветовое кодирование оценок
- 📱 Адаптивный дизайн

## 🛠 Технологии

- **Backend**: Python 3 + Flask
- **Frontend**: HTML, CSS, JS + Bootstrap 5
- **База данных**: JSON-файл
- **API**: REST endpoints

## 📁 Структура проекта
├── app.py # Основное приложение
├── grades.json # Хранение данных
├── requirements.txt # Зависимости
├── static/
│ └── style.css # Стили
└── templates/
├── index.html # Главная страница
└── grades.html # Страница оценок

## ⚙️ Быстрый старт


# 1. Клонировать и перейти в папку
git clone <repo-url>
cd journal-app

# 2. Установить зависимости
pip install flask

# 3. Запустить приложение
python app.py
